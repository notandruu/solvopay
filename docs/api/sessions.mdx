title: Sessions

## Register a session

```
POST /sessions
```

Called by the SDK after `SessionFactory.openSession` confirms on-chain. Records the session in the backend database.

**Request body:**

```json
{
  "session_id": "0x...",
  "escrow_address": "0x...",
  "agent": "0x...",
  "recipient": "0x...",
  "deposit": "10000000",
  "agent_yield_share": 80,
  "deadline_at": 1711929600
}
```

**Response `200`:**

```json
{
  "session_id": "0x...",
  "status": "ACTIVE",
  "created_at": "2024-04-01T00:00:00Z"
}
```

## Get session

```
GET /sessions/{session_id}
```

Returns session state and the latest submitted voucher.

**Response `200`:**

```json
{
  "session_id": "0x...",
  "escrow_address": "0x...",
  "agent": "0x...",
  "recipient": "0x...",
  "deposit": "10000000",
  "agent_yield_share": 80,
  "deadline_at": 1711929600,
  "status": "ACTIVE",
  "latest_voucher": {
    "total_authorized": "2000",
    "nonce": 3,
    "submitted_at": "2024-04-01T00:05:00Z"
  },
  "created_at": "2024-04-01T00:00:00Z"
}
```

`latest_voucher` is `null` if no vouchers have been submitted yet.

**Response `404`:**

```json
{ "detail": "session not found" }
```

## Trigger settlement

```
POST /sessions/{session_id}/settle
```

Instructs the backend to submit the latest voucher on-chain and settle the session.

The backend fetches the latest voucher, calls `SessionEscrow.settle(voucher, signature)`, and waits for the transaction to confirm.

**Response `200`:**

```json
{
  "tx_hash": "0x...",
  "status": "SETTLED"
}
```

**Response `400`:**

```json
{ "detail": "no vouchers submitted for this session" }
```

**Response `400`:**

```json
{ "detail": "session already settled" }
```
